Use this data source to query ECS instances by tag filters.

Example Usage

```hcl
data "edgenext_ecs_instance_tags" "example" {
  tag_key   = "env"
  tag_value = "dev"
  page_num  = 1
  page_size = 10
}

resource "edgenext_ecs_instance_tag" "binding" {
  instance_id   = data.edgenext_ecs_instance_tags.example.instance_tags[0].instance_id
  instance_name = data.edgenext_ecs_instance_tags.example.instance_tags[0].instance_name
  tag_ids       = [for t in data.edgenext_ecs_instance_tags.example.instance_tags[0].tags : t.id]
}
```

Argument Reference

* `tag_id` - (Optional) Tag ID filter.
* `tag_key` - (Optional) Tag key filter.
* `tag_value` - (Optional) Tag value filter.
* `page_num` - (Optional) Page number, default `1`.
* `page_size` - (Optional) Page size, default `10`.

Attributes Reference

* `id` - Stable query ID generated by provider.
* `total` - Matched resource count.
* `instance_tags` - Matched instance list. Common fields include `instance_id`, `instance_name`, `instance_type`, `region`, `tag_count`, and `tags`.
