Use this data source to query ECS images.

Example Usage

```hcl
data "edgenext_ecs_images" "example" {
  region     = "tokyo-a"
  visibility = "public"
  name       = "Debian"
  page_num   = 1
  page_size  = 10
}

output "first_image_id" {
  value = try(data.edgenext_ecs_images.example.images[0].id, null)
}
```

Argument Reference

* `region` - (Required) Region.
* `visibility` - (Optional) Image visibility, default `public`.
* `name` - (Optional) Image name filter.
* `status` - (Optional) Image status filter.
* `page_num` - (Optional) Page number, default `1`.
* `page_size` - (Optional) Page size, default `10`.

Attributes Reference

* `id` - Stable query ID generated by provider.
* `total` - Total image count in current query.
* `images` - Image list. Common fields include `id`, `name`, `description`, `status`, `visibility`, `size`, `os_distro`, and `os_version`.
